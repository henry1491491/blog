Python Django 資料庫（Schema）設定

  • model 的欄位
  • 欄位裡面的設定

model 的欄位

  • CharField 短中長度、長度固定的字串，必須設定最大長度

  • TextField 任意長度字串，最好設定最大長度限制（但只會作用在表單，資料庫不會去做限制）

  • IntegerField 儲存整數的欄位，在表格階段被驗證

  • DateField and DateTimeField 時間設定

目前還沒使用過

  • EmailField 儲存及驗證 E-mail 地址

  • FileField and ImageField 儲存上傳檔案跟圖檔（可以設定檔案或是照片的位置）

  • AutoField 一個自動增加的整數欄位。若無指定，會自動為你增加主鍵（Primary Key）

  • ForeignKey 外鍵，可對應到其它 models

  • ManyToManyField

還沒用過，等用過再來紀錄

更多類型參考：Model field reference | Django documentation | Django

欄位裡面的設定

可以設定在 models.CharField() 方法裡的參數：

  • help_text 創建一個 html 表單的 Label

  • verbose_name 用來給人閱讀的表單標籤名稱。如果沒設定會用 help_text 作為預設

  • default 每新增一筆紀錄，會自動創建的 field 的預設值

  • null

還沒使用，未來補充

  • blank 允許空格，與上方 null 設定有關

還沒使用，未來補充

  • choices

還沒使用，未來補充

  • primary_key 將這個欄位設為主鍵，如果沒有，Django 會自動幫你增加一個欄位為主鍵

更多設定參考：Model field reference | Django documentation | Django
